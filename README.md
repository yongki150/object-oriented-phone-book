## 클래스 다이어그램

- 객체가 실행되는 동안 대상과 연관이 있다면, Association(연관) 관계를 맺습니다.
- 객체의 메소드가 실행되는 동안만 대상과 연관이 있다면, Dependency(의존) 관계를 맺습니다.
