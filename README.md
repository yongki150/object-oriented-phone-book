## 클래스 다이어그램

- 객체 자체로 작동할 수 있다면, 대상과 Aggregation 관계를 맺습니다.
- 객체 자체로 작동할 수 없다면, 대상과 Composition 관계를 맺습니다.
- 객체 메소드가 실행되는 동안만 대상과 연관이 있다면, Dependency 관계를 맺습니다.
